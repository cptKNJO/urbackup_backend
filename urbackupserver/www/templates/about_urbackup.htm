Contribution of build server or testing infrastructure is welcome as well. The most appreciated contribution would be your time in form of help.

UrBackup is using following libraries/code:
  • The Crypto++ Library
  • The SQLite embedded database
  • cURL Library
  • Symas Lightning Memory-Mapped Database
  • FastCGI C++ Library
  • Mersenne Twister with improved initialization
  • MD5 implementation
  • MiniZ: ZIP archive reading/writing library
  • Templatized C++ Command Line Parser Library
  • jQuery JavaScript library
  • dust.js JavaScript templating library
  • JavaScript MD5 library
  • jqPlot: jQuery plotting plugin
  • DataTables: Table plug-in for jQuery
  • Bootstrap
  • Bootstrap flat theme
  • NProgress progress bar
  • Boostrap select plugin
  • Stanford Javascript Crypto Library
  • jQuery Sparklines
